HTML Tag List with Descriptions Tag Description Declares the document type and HTML version. Root element of an HTML document. Contains metadata, links, scripts, and styles. Sets the title of the document (appears in the browser tab). <meta> Provides metadata like charset, viewport, description, etc. <link> Links external resources like CSS files. <style> Embeds CSS styles inside HTML. <script> Embeds or links JavaScript code. <body> Contains the content of the webpage. Content & Structure Tag Description <header> Defines a header for a document or section. <nav> Declares a navigation menu. <main> Specifies the main content. <section> Defines a section in a document. <article> Contains self-contained content. <aside> Defines sidebar content. <footer> Declares a footer for a document or section. <div> Generic container for styling/layout. <span> Inline container for text/style purposes. Text Content Tag Description <h1> <h6> Heading tags from highest (<h1>) to lowest importance. <p> Paragraph text. <br> Line break. <hr> Horizontal rule (divider). <strong> Important text (bold). <em> Emphasized text (italic). <b> Bold text (styling only). <i> Italic text (styling only). <mark> Highlighted text. <small> Smaller text. <sub> Subscript text. <sup> Superscript text. <blockquote> Quoted block of text. <code> Inline code. <pre> Preformatted text. Links & Media Tag Description <a> Anchor tag for links. <img> Embeds an image. <audio> Embeds audio content. <video> Embeds video content. <source> Specifies multiple media resources. <iframe> Embeds another webpage. Lists Tag Description <ul> Unordered list (bullets). <ol> Ordered list (numbers). <li> List item. <dl> Description list. <dt> Term in a description list. <dd> Description of the term. Forms & Inputs Tag Description <form> Defines an HTML form. <input> User input field. <label> Label for form elements. <textarea> Multi-line input field. <button> Button element. <select> Drop-down list. <option> Option in a drop-down. <fieldset> Groups related form elements. <legend> Title for a <fieldset>. Other Useful Tags Tag Description <table> Table structure. <thead> Table header. <tbody> Table body. <tfoot> Table footer. <tr> Table row. <th> Table header cell. <td> Table data cell. <canvas> Used for drawing graphics via scripting. <svg> Scalable vector graphics. <noscript> Content for browsers with scripts disabled.
